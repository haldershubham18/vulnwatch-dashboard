# VulnWatch Dashboard

A React-based cybersecurity threat intelligence dashboard built with Vite.

![Dashboard](image.png)

## Tech Stack
- React 18 + Vite
- Context API (dark mode)
- Tailwind CSS
- NVD / NIST Public API

## Features
- Search CVEs by keyword (debounced)
- Filter by severity
- Sort by CVSS score (high to low / low to high)
- Pagination
- Dark mode toggle
